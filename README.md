## Optional requirements

- GNU/Make

## Usage

### Start

#### Makefile

```bash
make start
```

#### Docker Compose

```bash
docker-compose up --detach
```

### Stop

#### Makefile

```bash
make stop
```

#### Docker Compose

```bash
docker-compose down --remove-orphans --volumes --timeout 0
```

### Restart

#### Makefile

```bash
make restart
```

#### Docker Compose

```bash
docker-compose down --remove-orphans --volumes --timeout 0 && docker-compose up --detach
```
